# Valeur de Position - Interface Interactive

Une interface web interactive pour illustrer le concept de la valeur de position et du regroupement pour les nombres de 0 à 9999.

## 📚 Description

Cette application permet aux utilisateurs d'explorer visuellement :
- La décomposition des nombres en milliers, centaines, dizaines et unités
- Le concept de regroupement (10 unités = 1 dizaine, 10 dizaines = 1 centaine, 10 centaines = 1 millier)
- La représentation visuelle des différents ordres de grandeur

## 🚀 Utilisation

1. Ouvrez le fichier `index.html` dans un navigateur web
2. Entrez un nombre entre 0 et 9999 dans le champ de saisie
3. Ou utilisez les boutons :
   - **🎲 Aléatoire** : Génère un nombre aléatoire
   - **🔄 Réinitialiser** : Remet le compteur à 0

## ✨ Fonctionnalités

- **Visualisation interactive** : Représentation visuelle des unités, dizaines, centaines et milliers
- **Décomposition mathématique** : Affichage des formules de décomposition
- **Concept de regroupement** : Illustrations des échanges entre ordres de grandeur
- **Interface responsive** : Fonctionne sur ordinateur, tablette et mobile

## 🛠️ Technologie

- HTML5
- CSS3 avec animations et gradients
- JavaScript vanilla (aucune dépendance externe requise)

## 📖 Concept pédagogique

L'interface illustre le système de numération décimale où chaque position vaut 10 fois plus que la position à sa droite. C'est le principe fondamental du regroupement : 10 éléments d'un ordre peuvent toujours être échangés contre 1 élément de l'ordre supérieur.
